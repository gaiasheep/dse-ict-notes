Key Concepts

  • Primary Key (ä¸»é—œéµç¢¼): A field that uniquely identifies each record. Cannot be NULL. (e.g., StudentID).
  • Foreign Key (å¤–éµç¢¼): A field that links to the Primary Key of another table.

SQL Cheat Sheet (SQL é€ŸæŸ¥è¡¨)

Based on the provided notes, here is the standard structure:

SELECT Field1, Field2, ...    -- Specify columns (DISTINCT removes duplicates)
FROM TableName                -- Specify table
WHERE Condition               -- Filter records (e.g., Age > 18)
GROUP BY Field                -- Group records for calculation
ORDER BY Field [ASC/DESC]     -- Sort results (ASC=Ascending, DESC=Descending)

Common Operators & Functions

  • Text Matching: LIKE 'K%' (Starts with K), LIKE '%A%' (Contains A).
    • Note: In DSE, % is the wildcard for multiple characters, _ is for a single character.
  • Range: BETWEEN 10 AND 20
  • Null Check: IS NULL, IS NOT NULL
  • Aggregate Functions (çµ±è¨ˆå‡½æ•¸):
    • COUNT(*): Count all records.
    • SUM(Field): Total sum.
    • AVG(Field): Average.
    • MAX(Field) / MIN(Field)
